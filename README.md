# SillyTavern Translate Lang

Расширение для [SillyTavern](https://github.com/SillyTavern/SillyTavern) — интерфейса для работы с LLM, позволяющее переводить сообщения прямо в чате с сохранением форматирования.

Плагин добавляет кнопку перевода к сообщениям ассистента и позволяет:
- переводить текст через подключённый LLM (KoboldCPP / textgen и др.)
- сохранять оригинал через swipe
- просматривать результат перед применением

---

## 📦 Установка

1. Перейдите в Extensions в SillyTavern  
2. Установите расширение через Git URL:
```
https://github.com/trijin/sillytavern-translate-lang
```

3. Перезапустите SillyTavern (если требуется)

---

## ⚙️ Настройки

![Settings](https://trijin.github.io/images/sillytavern-translate-lang/settings.png)

### Целевой язык
Язык, на который будет выполняться перевод.

### Проверять наличие текста для перевода
Если включено:
- плагин пытается определить, есть ли в сообщении текст, требующий перевода
- если текст уже на целевом языке — перевод не выполняется

Если выключено:
- перевод будет выполняться всегда

### Промпт перевода
Текст промпта, который отправляется в модель.

Если поле **пустое** — используется встроенный дефолтный промпт (тот, что показан в интерфейсе).

Поддерживаемые плейсхолдеры:
- `{{language}}` — заменяется на выбранный язык

## 🌐 Кнопка перевода

![Translate Button](https://trijin.github.io/images/sillytavern-translate-lang/button.png)

В каждом сообщении ассистента появляется кнопка перевода: 🌐 **Иконка перевода**

При нажатии:
- текст сообщения отправляется в модель
- открывается окно предпросмотра, с задержкой, в зависимости от скорости модели.

---

## 🪟 Окно перевода

![Preview](https://trijin.github.io/images/sillytavern-translate-lang/preview.png)

После перевода появляется окно с двумя колонками:
- слева — оригинал
- справа — перевод

### Доступные действия

- **Применить**  
  Заменяет сообщение на перевод  
  Оригинал сохраняется как swipe

- **Закрыть**  
  Отменяет перевод

- **Повторить**  
  Выполняет перевод ещё раз  
  Полезно, если модель выдала неудачный результат

---

## 🧠 Особенности

- Оригинальный текст не теряется (через swipe)
- Используется текущий textgen backend
- Перевод выполняется без истории чата (изолированный запрос)

---

## ⚠️ Ограничения

- Качество перевода зависит от используемой модели
- Проверка языка работает не для всех языков
- HTML/разметка отображается как есть (для сохранения структуры)
